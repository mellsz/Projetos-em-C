# 💻 C & C++ Programming-Vault
> "C is quirky, flawed, and an enormous success." — Dennis Ritchie

Bem-vindo ao meu baú de projetos! Este repositório documenta minha evolução nas linguagens C e C++, explorando desde a lógica básica até conceitos de Orientação a Objetos e gestão de memória.

## 📚 Estrutura do Repositório

📂 **/Estruturas_Dados:** Listas ligadas, pilhas, filas e árvores (Implementações em C e C++).

📂 **/Algoritmos_Ordenacao:** QuickSort, BubbleSort, MergeSort, etc.

📂 **/Sistemas_Basicos:** Projetos práticos como Calculadoras, Calendários e conversores.

📂 **/Sistemas_Complexos:** Pequenos softwares (ex: Gerenciador de estoque, Cadastro de clientes).

## ⚙️ Como compilar e rodar

Dependendo da extensão do arquivo **(.c ou .cpp)**, utilize o compilador correspondente:

**1. Para arquivos em C**

```Compilar```
```gcc nome_do_arquivo.c -o executavel```

```Executar```
```./executavel```

**2. Para arquivos em C++**

```Compilar```
```g++ nome_do_arquivo.cpp -o executavel```

```Executar```
```./executavel```
